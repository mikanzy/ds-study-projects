# Учебные проекты по Data Science

Данные проекты были выполнены в ходе обучения в **Яндекс.Практикуме** на курсе «Специалист по Data Science». Далее в таблице проекты представлены в хронологическом порядке — от первых шагов в анализе данных до заключительных проектов полного цикла построения и анализа ML моделей.

| № | Название проекта | Описание | Используемые библиотеки | 
| :---: | :--- | :--- | :--- |
| 1 | [Музыка больших городов](01_big_cities_music) | Проведено сравнение предпочтений пользователей Яндекс.Музыки из Москвы и Санкт-Петербурга в зависимости от времени (утро и вечер) и дня недели (понедельник, среда, пятница). | *pandas* |
| 2 | [Исследование надёжности заёмщиков](02_borrowers_reliability) | Исследовано влияние признаков, характеризующих заёмщика, на возврат им кредита в срок. Построены портреты надёжного и рискованного заёмщика. На основании найденных закономерностей сформулированы рекомендации, которые помогут при построении модели кредитного скоринга. | *matplotlib, pandas* |
| 3 | [Исследование объявлений о продаже квартир](03_real_estate) | Определены параметры, влияющие на цену квартиры, и степень их влияния. Установлены принципы определения рыночной стоимости объектов недвижимости в зависимости от их параметров. Сформулированы критерии, по которым можно выявить объекты с аномальной стоимостью по отношению к рыночной. | *matplotlib, numpy, pandas* |
| 4 | [Анализ данных сервиса аренды самокатов](04_scooter_rental) | Проанализированы данные о пользователях сервиса и их поездках с акцентом на сравнении пользователей с подпиской и без подписки. Проверены статистические гипотезы о длительности поездок пользователей и выручке сервиса от них. Даны ответы на поставленные вопросы, связанные с планированием маркетинговых акций. | *matplotlib, numpy, pandas, scipy* |
| 5 | [Анализ продаж компьютерных игр](05_video_game_sales) | Исследованы данные о продажах игр, оценках пользователей и экспертов, жанрах и платформах. Выявлены факторы, определяющие успешность игры в продажах. Составлен портрет пользователя для каждого региона. Проверены статистические гипотезы. Сформулированы рекомендации, которые помогут интернет-магазину компьютерных игр спланировать рекламные кампании на следующий год. | *matplotlib, numpy, pandas, scipy* |
| 6 | [Разработка прогнозирующих моделей для молочного хозяйства](06_dairy_farm) | Разработаны ML модели, которые помогут фермеру принимать решение о покупке коров для молочного хозяйства: модель для прогнозирования удоя и модель для предсказания того, будет ли вкусным молоко. На основании прогнозов созданных моделей отобраны подходящие коровы для покупки. | *matplotlib, numpy, pandas, phik, scikit-learn, seaborn, statsmodels* |
| 7 | [Удержание активности клиентов интернет-магазина](07_online_store_customer_activity) | Разработано решение для удержания покупательской активности постоянных клиентов интернет-магазина. Построена ML модель, которая предсказывает вероятность снижения покупательской активности клиента в следующие три месяца. Используя данные модели и данные о прибыльности клиентов, выделен один из сегментов покупателей. Для выделенного сегмента разработаны персонализированные предложения. | *matplotlib, numpy, pandas, phik, scikit-learn, seaborn, shap* |
| 8 | [HR-аналитика](08_hr_analytics) | Разработаны ML модели для прогнозирования уровня удовлетворённости и предсказания увольнения сотрудников, которые помогут HR-отделу компании спрогнозировать текучку кадров. Составлен характерный портрет уволившегося сотрудника. Сформулированы рекомендации по оптимизации кадровой политики в компании. | *matplotlib, numpy, optuna, pandas, phik, scikit-learn, scipy, seaborn, shap* |
| 9 | [Выбор локации для скважин](09_oil_wells_location) | С использованием ML модели выбран регион для бурения новых скважин, где добыча нефти принесёт наибольшую прибыль. | *matplotlib, numpy, pandas, scikit-learn, seaborn* |
| 10 | [Определение стоимости автомобилей с пробегом](10_car_price) | Построена ML модель для определения стоимости автомобиля по его характеристикам. | *catboost, lightgbm, matplotlib, numpy, optuna, pandas, phik, scikit-learn, seaborn, shap* |
| 11 | [Прогнозирование заказов такси](11_taxi_orders) | Построена ML модель для предсказания количества заказов такси в аэропортах на следующий час. | *lightgbm, matplotlib, numpy, pandas, scikit-learn, seaborn, shap, statsmodels* |
| 12 | [Определение токсичных комментариев](12_toxic_comments) | Построена ML модель для классификации комментариев на позитивные (нормальные) и негативные (токсичные). | *langid, lightgbm, matplotlib, numpy, pandas, regex, scikit-learn, seaborn, spacy, tqdm* |
| 12a | [Определение токсичных комментариев с использованием BERT](12a_toxic_comments_bert) | Попытка построить модель классификации комментариев на позитивные (нормальные) и негативные (токсичные) с использованием модели BERT для извлечения признаков (feature extraction) из текстов. Построенная модель не достигла целевого уровня качества. | *catboost, langid, matplotlib, numpy, pandas, scikit-learn, seaborn, shap, torch, tqdm, transformers* |
| 13 | [Определение возраста покупателей магазина по фотографиям](13_age_by_photo) | Построена нейронная сеть, которая по фотографии определяет возраст человека. | *matplotlib, numpy, pandas, PIL, scikit-learn, scipy, seaborn, tensorflow* |
| 14 | [Имитация технологического процесса обработки стали на металлургическом комбинате](14_steel_mill) | Разработана ML модель для прогнозирования температуры розлива сплава на этапе обработки стали, которая поможет металлургическому комбинату оптимизировать производственные расходы. На основе полученной в результате имитационного моделирования информации сформулированы рекомендации для снижения потребления электроэнергии. | *catboost, lightgbm, matplotlib, numpy, optuna, pandas, scikit-learn, seaborn, shap, statsmodels* |
| 15 | [Предсказание оттока клиентов оператора связи](15_telecom_customer_churn) | Разработана ML модель для прогноза оттока клиентов, которая предсказывает, разорвёт ли абонент договор, на основе персональных данных о клиенте, а также информации о тарифе и используемых услугах. Выявлены факторы, определяющие склонность клиентов к оттоку. Построен портрет типичного уходящего клиента, сформулированы вероятные причины оттока клиентов и рекомендации по удержанию уходящих клиентов. | *catboost, matplotlib, numpy, optuna, pandas, phik, scikit-learn, seaborn, shap* |
